# Cybersecurity Portfolio

Collection of hands-on cybersecurity work covering both offensive and defensive security. All activities were conducted in controlled lab environments.

---

## Overview

This repository documents practical experience in:
- Web application security testing
- Network analysis and exploitation
- Security monitoring and detection engineering
- Secure infrastructure design

---

## Offensive Security

### Web Exploitation
- SQL Injection (manual + automated with sqlmap)
- Cross-Site Scripting (reflected, stored, DOM-based)
- Cross-Site Request Forgery (token bypass techniques)
- Command Injection and File Inclusion

### Network Attacks
- Nmap scanning (TCP, SYN, UDP)
- Man-in-the-Middle (ARP spoofing)
- WiFi attacks (handshake capture and cracking)

### Credential Attacks
- Brute-force with Burp Suite Intruder
- Wordlist attacks using wfuzz and SecLists

---

## Defensive Security

### Network Security
- Segmented network design using pfSense
- VLAN-based isolation (LAN, DMZ, WAN)

### Detection & Monitoring
- Wazuh SIEM deployment and alerting
- Suricata IDS with custom rule creation
- Zeek + ELK stack for traffic analysis

### Infrastructure Security
- Kubernetes deployment with monitoring (Prometheus/Grafana)
- Web Application Firewall (ModSecurity)
- VPN setup using OpenVPN
